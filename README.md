# 研究生纪念网

这里是一个面向公众的缅怀页面。

我们希望用真实、克制、尊重的方式，记录那些离开的研究生，记住他们的名字、努力与人生片段。  
每一位被展示的人，都不应只被一个结局定义。

## 建站初衷

在学业、就业、经济、家庭与社会压力交织的现实里，很多研究生承受着外界难以看见的负担。  
这个页面的存在，是为了纪念，也是为了提醒：

- 研究生不是统计数字
- 每一次沉默都值得被倾听
- 每一个名字都值得被认真对待

## 我们希望留下的内容

- 个人姓名与照片
- 生平与学习经历的简要介绍
- 同学、朋友、访客的缅怀留言

## 留言倡议

欢迎留言，但请共同维护基本边界：

- 保持尊重，不攻击、不嘲讽
- 不传播未经核实的隐私与细节
- 不消费他人的伤痛，不将悲剧娱乐化

## 关于维护

人物资料由站点维护者统一整理后发布。  
评论区向公众开放，便于更多人表达纪念与关怀。

## 写在最后

愿逝者安息。  
愿仍在路上的人，被看见，被理解，被善待。

如果你或你身边的人正处在明显的心理危机中，请优先联系当地紧急救助渠道、学校心理中心或可信任的亲友，尽快获得现实支持。
